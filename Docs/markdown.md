## Archivos Markdown (`.md`, `.markdown`)

Al abrir o crear un archivo Markdown, el editor muestra solo el **texto fuente** por defecto. La vista previa renderizada es **opcional**.

### Abrir / cerrar la vista previa

- Botón **👁️ Vista previa** encima del editor (en pestañas `.md`)
- Menú **Vista → Vista previa Markdown**
- Atajo **`Ctrl+Shift+V`**

La previsualización se actualiza mientras escribes (solo cuando está visible).

### Ejecución de código

En archivos **`.md`** y **`.txt`** el botón **Ejecutar Código** y el atajo **`Ctrl+Enter`** están **deshabilitados**: no son ejecutables como Python.

## Archivos de texto (`.txt`)

Se abren en el editor con resaltado básico. Tampoco permiten ejecutar código con `Ctrl+Enter`.

## Guardado

Los archivos `.md` y `.txt` se guardan con **Archivo → Guardar** (`Ctrl+S`) o **Guardar como…** (`Ctrl+Shift+S`), igual que el resto de formatos soportados.
