# Añadir tutoriales

- [← Índice](./README.md)
- [Portada](../README.md)

Los tutoriales se definen en `tutorials_config.py`.

## Pasos
1. Abre `tutorials_config.py`
2. Añade un nuevo tutorial (id, título, descripción, dificultad)
3. Añade pasos con:
   - texto/explicación
   - código de ejemplo
   - validación (si aplica)

## Consejo
Mantén los ejemplos pequeños y validables para que el flujo educativo sea fluido.
