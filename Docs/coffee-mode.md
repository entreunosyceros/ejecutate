# Modo Café ☕

- [← Índice](./README.md)
- [Portada](../README.md)

## Qué hace
- Cambia a un tema cálido (“Café”)
- Muestra un overlay centrado con una taza humeante
- **Bloquea toda la UI** hasta que el usuario mueva el ratón o pulse una tecla

## Personalización
- Preferencias → **☕ Café**: define el mensaje mostrado en el overlay
- Atajo: `Ctrl+Alt+C`
