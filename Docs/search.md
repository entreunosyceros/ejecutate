## Search (sidebar)
- Resultados clicables con **preview** del match
- Flags:
  - **Aa**: case sensitive
  - **Ab**: palabra completa
  - **.\***: regex
  - **En archivos**: buscar en el root del explorador
- Replace:
  - **Reemplazar** / **Reemplazar todo**
  - En modo “En archivos”, *Reemplazar todo* aplica cambios a archivos del proyecto (respetando patrones/exclusiones).
